## About

FOODSTEM is a desktop application for restaurants and fast-food businesses. It allows cashiers to process orders and managers to handle inventory.

---

## Features

### Login & Register
- Create account with name, phone, username, password
- Login with username and password

### POS (Point of Sale)
- Browse products by category (Burger, Wrap, Fries, Soda, Dessert, Sandwich)
- Add products to cart
- Enter amount and calculate change
- Pay and update stock automatically

### Inventory Management
- View all products
- Add new products
- Edit existing products
- Delete products
- Search products

---

## 🛠️ Technologies

- Java 8+ (Swing)
- JDBC
- MySQL
- NetBeans IDE
- MySQLWorkbench
